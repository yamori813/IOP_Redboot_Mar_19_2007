dcsr

Name

dcsr -- Dump/modify the current DCSR

Synopsis

dcsr [-h Hot Debug on] [-c Hot Debug off] [-t Trace on] [-n Trace off]

Description

Displays and/or modifies the current value of the DCSR.

Examples

Dump the current DCSR. 
 
RedBoot> dcsr
dcsr = 0x8000001c
